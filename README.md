# Datasheet Search

A unified electronic component datasheet search tool aggregating multiple supplier databases in one convenient interface.

## Features

- **Multi-source Search**: Query across NEXAR and TrustedParts simultaneously
- **Component Discovery**: Browse by category (Voltage Regulators, Op-Amps, Microcontrollers, and more)
- **Popular Components**: Quick access to frequently used components
- **Search History**: Automatic tracking of recent searches for quick reference
- **Bookmarks**: Save favorite components for future access
- **Quick Filters**: Pre-built filter buttons for common component types:
  - Voltage Regulators
  - Operational Amplifiers
  - Microcontrollers
  - Logic ICs
  - Memory devices
  - And more
- **Comprehensive Details**: View specifications, pricing, availability, and technical information
- **Offline Support**: Search history and bookmarks accessible offline

## Tech Stack

- **HTML5** - Semantic markup
- **JavaScript** - Core application logic and API integration
- **Web APIs** - Fetch API for datasheet source integration
- **Local Storage** - Bookmarks and search history management
- **PWA** - Progressive Web App capabilities

## Getting Started

### Online Demo
Visit the live application: [https://arnoutzw.github.io/datasheet-search-app/](https://arnoutzw.github.io/datasheet-search-app/)

### Local Installation
Simply open `index.html` in a modern web browser. No build tools or dependencies required.

### Usage
1. Open the application in your browser
2. Enter a component name or part number in the search box
3. Browse results from multiple sources side-by-side
4. Use quick filter buttons for common component types
5. Bookmark favorite components for quick access
6. View search history from previous sessions

## Browser Support

Requires a modern browser with Fetch API support (Chrome, Firefox, Edge, Safari)
